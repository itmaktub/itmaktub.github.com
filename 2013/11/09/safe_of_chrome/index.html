    　　那天在机房上课的时候，发现屏蔽广告的插件没有自动同步，就自己去商店下载了一个，着急联网，没细看，然后自己回来上网的时候，一不小心就把它当正主了，今天才发现，访问谷歌首页的时候会有一个图标，显示有恶意脚本，打开开发者模式，发现有下载5ihaitao.com的脚本，cookie中也包含这个脚本，尼玛大谷歌会在网站中包含这脚本？立马怀疑自己中标了，把所有插件禁止，然后再打开，同时刷新谷歌首页，终于找到原来是“广告终结者”这款插件，赶紧去商店里看看评论，尼玛啊，就二十多个人下载，两个人评论说好，而且目前已经被商店停用了，次奥，我怎么这么粗心！网上搜索一番。发现很多恶意插件都和这个51haitao有关系，当你购物时劫持你的流量到返利网站，以此获取利益，马蛋，你们为什么这么坏，这下涨记性了，以后安装chrome插件的时候都小心点，看看评论，人数，作者等等，一般作者用很牛逼的很出名的公司（中文）的，大部分都有问题，评论很少，又都说好的，没有一点反馈信息的，都有猫腻。chrome商店的审查也太不给力了，全是重名字的，眼花缭乱，希望大家擦亮双眼，以此为戒。

    如何向 Google Chrome 团队举报

    Chrome Web Store 现阶段还没有有效的阻止恶意扩展的办法，暂时只能靠用户来主动举报而采取处理。

    另外因为 Google 工作人员有限，可能不能及时地处理举报信息。

    于是推荐大家见到恶意扩展都主动去举报下，反馈多了也许才能引起重视。举报恶意扩展的步骤：

    1. 在每个扩展程序主页的详细介绍页面都有一个 Report Abuse 的链接

    2. 填写表格向 Google 反馈

    选项可以选择?<span style="color: #333333;"><span style="font-family: Consolas, 'Liberation Mono', Courier, monospace;"><span>This item contains malware or engages in harmful activity.</span></span></span>

    然后反馈的内容可以酌情填写，例如<span style="color: #333333;"><span style="font-family: Consolas, 'Liberation Mono', Courier, monospace;"><span>It downloads a malicious javascript code from a remote server which steals and modifies users' cookies.</span></span></span>

    甄别恶意扩展的常用办法

    1. 要求不必要的权限

    恶意扩展一般都要求一些跟功能无关的权限，用于其恶意行为。

    2. 制作粗糙

    恶意扩展的制作都比较粗糙，没有完善的图标和介绍说明等。

    3. 冒充官方帐号或者程序名称

    一般冒充官方程序名称的（例如优酷海外版），或者开发者冒充官方帐户的，程序基本上都是有问题的。

    Google 官方的扩展程序

    冒充 Google 的恶意程序

    4. 用户人数差别

    原版的安全的扩展程序一般用户都要比恶意扩展程序多一些，因为原版扩展的存在时间更久，并且了解技术的用户也会主动选择。所以选择用户人数多的扩展程序一般都不会有问题。

    5. 查看用户评论

    强烈推荐安装扩展程序之前浏览一下现有用户的评论，特别是 Recent/最新 的评论。

    不过有最近别的扩展遭受马甲或者水军的恶意评论，详见受害的开发者的原文。

    0xFF. 直接检查源码

    以上的都是只看表面的办法，不保证完全靠谱。懂技术的同学可以直接检查扩展程序的源代码。

    查看在 Chrome 后台运行的扩展程序的源码可以访问 chrome://extensions 页面，勾选上 Dev mode，然后点击相应扩展程序的 Inspect views 后面的链接就可以看到源码了。

    想不安装扩展程序而直接检查源码的话，可以利用一些工具，例如?http://crx.zhuzhu.org，下载 crx 安装文件至本地，然后 unzip 解压缩就可以直接看到所有的源代码了。
