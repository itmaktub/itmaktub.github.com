树莓派上手试玩

By itmaktub

2013-11-03 Updated:2015-06-12

Contents

树莓派终于到了，第一眼看到它，唯一的感觉就是小，小巧玲珑啊，一只手就可以完全握住。慌忙吃过饭，赶紧开始折腾吧。

  1. 下载官方推荐的raspbian

http://downloads.raspberrypi.org/raspbian_latest

（500多兆，渣网速下载了一个多小时我会乱说？）

  1. 解压出img镜像

  2. 用win32diskimager将镜像写入SD卡

  3. 上电开机，用路由器连接树莓派，查看dhcp服务器列表，找到树莓派的IP地址

  4. 打开putty，输入IP地址，连接树莓派

  5. 执行sudo apt-get install tightvncserver，下载并设置tightvncserver

  6. 执行sudo raspi-config配置图形界面及语言、使用整张SD卡等

  7. 电脑端打开VNC-Viewer，输入IP，输入密码访问图形桌面，详细的安装可以看这里：

本来的顺序大概就这些，但在我这里遇到的最大的困难就是没有网络，当然手机可以访问互联网。用手机给电脑共享网络，企图使树莓派联网，折腾半天没搞定，最后还是祭出了大杀器——无线网卡，嘿嘿，一下子就连接上网了，
